## KEY ENHANCEMENTS

### 1. Tier-Based Organization
**Original:** Flat list of 8 domains
**Enhanced:** 3-tier hierarchy
- **Tier 1 (30%):** Foundation-level macro risks (geopolitics, finance, compliance)
- **Tier 2 (35%):** Operational execution risks (supply chain, cybersecurity, HR)
- **Tier 3 (35%):** Organizational/human factors (founder psychology, reputation, ESG)

**Why This Matters:** Allows stakeholders to understand risk at different strategic levels. Founders focus on Tier 3, CFOs on Tier 1, COOs on Tier 2.

### 2. Polycrisis Modeling
**Original:** Independent domain scoring
**Enhanced:** Explicit "polycrisis convergence" domain + multiplier
- If 5+ domains >60: apply 1.15x penalty
- Recognition that 2026 risks don't operate in isolation
- Multiple simultaneous shocks overwhelm traditional mitigation

- EU AI Act enforcement (€35M fines or 7% turnover)

## SCORING SOPHISTICATION COMPARISON

### Original Algorithm
```
domain_score = 100 - (average_response × 20)
weighted_total = Σ(domain_score × weight)
final_score = weighted_total × stage_multiplier
```

### Enhanced Algorithm
```
// Step 1: Domain scores
domain_score = 100 - (average × 20)

// Step 2: Flag detection
if min(responses) ≤ 1: RED_FLAG (+50)
elif min(responses) ≤ 2 AND count(≤2) ≥ 2: ORANGE_FLAG (+30)
elif average ≤ 2.5: YELLOW_FLAG (+15)

// Step 3: Critical domain amplification
if flag in [Founder Risk, Cybersecurity, AI Compliance]:
  flag_penalty *= 1.5

// Step 4: Weighting & adjustments
weighted = Σ(domain_score × domain_weight)

// Step 5: Multipliers
stage_multiplier = {pre-seed: 1.4, seed: 1.2, series-a: 1.0, series-b: 0.85, series-c+: 0.7}
vertical_multiplier = {fintech: 1.3, healthtech: 1.25, saas: 1.0, consumer: 0.9, hardware: 1.1}
polycrisis = count(domains > 60) >= 5 ? 1.15 : 1.0

// Step 6: Final calculation
final_score = weighted × stage_multiplier × vertical_multiplier × polycrisis
final_score = min(final_score, 100)
```

**Why Enhanced is Better:**
- Captures nuance (flags vs. domain average)
- Contextualizes by stage (pre-seed riskier than series C)
- Recognizes industry vertical (fintech more regulated)
- Models polycrisis (multiple shocks amplify)
- Amplifies critical domains (psychological + security + regulatory)
